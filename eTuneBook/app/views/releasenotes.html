Release Notes


eTuneBook Updates are anounced at eTuneBook on FB.


1.1.8 - 27.11.13

  • Note Editor: Edit Note elements by choosing from options instead of typing text: Chord, Fingering, Accidental, Note, Octave and Length.

1.1.7 - 12.11.13

  • Export TuneBook to Google Drive.
  • Import TuneBook from Google Drive.
  • Loading Bar during HTML Requests.
  • Fixed broken Video.

1.1.6 - 18.10.13

  • Optimization for small screens: ngTouch for faster clicks.
  • Same Filter for Tunes and Sets.
  • Filter for a given range of playing frequency.
  • Filter for a given range of modification dates.
  • Filter for a given range of play dates.
  • Picking date ranges by bootstrap-daterangepicker.
  • 'Just played the Set' can now be pressed directly on the Set List.

1.1.5 - 10.10.13

  • Search-Box on Tune-Set-List (although not refreshing yet upon removing the search text again. work in progress...).
  • Search-Box on Tune-List (skipped ng-grid column menu).
  • Navigating back from a Set: Set should now be first set on list (buz id not in most cases. work in progress...).
  • Navigating back from a Tune: Tune is now first tune on list.
  • Buttons: YouTube, FaceBook, Twitter.
  • Optimization for small screens: Less Tune-Set-List-Columns on mobile Devices.
  • Optimization for small screens: Less Tune-List-Columns on mobile Devices.
  • TuneBook was cached by Browser which prevented loading of new Versions. Cache removed from Ajax-Request.
  • Creating a new Set by copying/moving a Tune (by not choosing a Target).
  • Moving Tunes within same Set was broken. Fixed now.

1.1.4 - 29.09.13

  • Random Selection of Tunes and Sets: Filter by Playdates.
  • Tune-View: Sets-Selection navigates away to Set-View, if only one Set exists.
  • Tune- and Set-List: Links to Record-Views by Standard Anchors.
  • Set-List: Link to Tune-View.
  • Moving/Copying Tunes was broken. Fixed now.

1.1.3 - 16.09.13

  • Random Tune in Tune-View, Random Set in Set-View.
  • Tune-Selection in Tune-View, Set-Selection in Set-View.
  • Optimization for small screens: Tune Dropdown Button instead of Tabs.
  • Optimization for small screens: NavBar Buttons, non-collapsing.
  • Optimization for small screens: BookTitle removed from Navbar.

1.1.2 - 12.09.13

  • Optimization for small screens: Tune-Set-Filter moved from bottom-fixed NavBar to a Modal Dialog.
  • Optimization for small screens: BookTitle moved to Navbar.
  • Fixed broken Modals in Set-View (Bootstrap Migration Part 2).

1.1.1 - 09.09.13

  • Fixed broken Navigation from the Set-View to the Tune-View in Firefox and IE.
  • Angular-UI-Router Version 0.2.0
  • Angular Version 1.0.8. Removed Animation Stuff.
  • Bootstrap Version 3.0.0. Fix for broken Navigation on Smartphones.

1.1.0 - 31.07.13

  • Major Rework of Navigation (migrated to ui-router): Tune List, Set List, Tune Editor and Set Editor.
  • Top Navigation simplified to: Book, Sets, Tunes, Info.
  • Edit Book Title, Version and Description (Book Editor).
  • Introduction of fast scrolling List Grids (migrated to ng-grid).
  • New Tune List (ng-grid) supporting fast sorting, filtering and grouping.
  • Set-Editor: Sorting TuneSet Position as Number (sets having more than 10 Tunes are now sorted as they should)
  • Dropped support for offline modus (appcache skipped).
  • Temporarily dropped Midi Support due to problems with the Quicktime-Plugin.
    Midi-Support will be re-introduced as soon as abcjs will support javascript Midi.
  • Major Rework of how videos and external sites ar managed (ng-grid). You can now add more than 3 records.
  • 'Just played the set' puts the current date and time on all tunes in a set.
  • 'Just played the tune' can be clicked in the Set Editor and the Tune Editor.
  • Filter sets for Target, Environment and Name of TuneSet.
  • Delete a set position.
  • Add a new Set for an existing tune.
  • Add a comment specific for a set position.
  • Development/Deployment: Migrated to WebStorm and Yeoman.

1.0.7 - 23.05.13

  • Fixed New TuneBook/New Tune, which was broken when no TuneBook was imported before.
  • Export TuneBook to a File on your hard disk.
  • Edit TuneSet Infos: Target, Environment and Name of TuneSet.
  • Fixed broken dropdown when navigation bar was collapsed
  • Copy a tune to another set
